# Практична робота №5: Система керування користувачами (User Management System)

## 📌 Опис проєкту
Цей проєкт демонструє реалізацію системи керування даними користувачів з використанням сучасних стандартів JavaScript (ES6+). Основна мета — показати вміння працювати з безпекою даних, валідацією та імутабельністю.

## 🛠 Використані технології
* **HTML5 / CSS3 (Bootstrap 5)** — для створення адаптивного інтерфейсу.
* **JavaScript (ES6+)** — основна логіка додатка.

## 🚀 Ключові концепції, реалізовані в роботі:

1. **Інкапсуляція та замикання (Closures):**
   Вся логіка та масив даних приховані всередині модуля `UserSystem`. Доступ до даних з глобальної області видимості (консолі) закритий, що запобігає несанкціонованій зміні бази користувачів.

2. **Валідація через Regular Expressions (RegExp):**
   Реалізована перевірка вводу імені. Система блокує цифри та некоректні символи, дозволяючи лише літери (від 2 до 20 знаків).

3. **Імутабельність (Immutability):**
   При оновленні даних або додаванні нових користувачів ми не змінюємо існуючий масив або об'єкти напрямую. Використовуються методи `.map()` та spread-оператор (`...`) для створення нових станів даних.

4. **Методи масивів ES6+:**
   Для роботи з даними використані методи `map`, `some`, `filter` та `spread`, що робить код лаконічним та зрозумілим.

## 📸 Як перевірити роботу:
1. Відкрийте `index.html` у браузері.
2. Відкрийте консоль розробника (**F12**).
3. Спробуйте додати користувача з некоректним ім'ям (наприклад, "Дмитро7").
4. Спробуйте змінити роль користувача за його ID.
5. Переконайтеся, що вивід у консолі відбувається у форматі зручної таблиці (`console.table`).
