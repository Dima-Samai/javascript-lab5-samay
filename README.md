# User Management System (JS Objects Practical)

## Реалізовані вимоги:
- **UserManager (Closure):** Дані користувачів повністю захищені від прямого доступу.
- **User Object:** Містить методи `getInfo()` та `isAdmin()` з використанням `this`.
- **ES6+:** Деструктуризація параметрів, Template literals, Arrow functions.
- **Імутабельність:** Операції оновлення та додавання не змінюють старі об'єкти (Spread & Map).
- **Валідація:** Використання регулярних виразів для перевірки імен та email.
- **Unit Tests:** Вбудована система тестів з автоматичним звітом у консоль.
- **JSDoc:** Весь код задокументований згідно зі стандартами.

## Як запустити:
1. Відкрийте `index.html` у браузері.
2. Відкрийте консоль (F12) для перегляду результатів тестів.
