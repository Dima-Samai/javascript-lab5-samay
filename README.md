# Система управління користувачами (User Management System)

Цей проект реалізований в межах практичної роботи №5 з дисципліни "Основи програмування мовою JavaScript".

## Структури даних
- **User Object**: містить id, name, email, role та методи getInfo(), isAdmin().
- **UserManager**: використовує замикання (closures) для безпечного зберігання масиву користувачів.

## Функціонал
- Створення, видалення та оновлення користувачів.
- Валідація email за допомогою регулярних виразів.
- Імутабельні операції (Immutable operations) при зміні даних.
- Пошук за ID та фільтрація за роллю.

## Як запустити
1. Завантажте файли.
2. Відкрийте `index.html` у браузері.
3. Результати тестів відображаються у консолі розробника (F12).
